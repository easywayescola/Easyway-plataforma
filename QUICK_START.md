# 🚀 Guia Rápido - EasyWay

Bem-vindo à EasyWay! Este guia rápido ajudará você a começar em minutos.

## 👤 Para Alunos

### 1️⃣ Fazer Login

1. Acesse a página inicial
2. Clique em **"Entrar como Aluno"**
3. Use as credenciais:
   - **Email**: aluno@easyway.com.br
   - **Senha**: 123456

### 2️⃣ Explorar o Dashboard

Você verá:

| Seção | O que fazer |
|-------|-----------|
| **Estatísticas** | Veja seus pontos, nível e progresso |
| **Super Skills** | Aprenda habilidades socioemociais |
| **Livro Interativo** | Leia histórias em inglês |
| **Níveis** | Pratique quizzes (A1, A2, B1) |
| **Ranking** | Veja como você está em relação aos colegas |

### 3️⃣ Começar a Aprender

**Opção A: Super Skills**
- Clique em "Explorar Super Skills"
- Escolha uma lição (Confidence, Kindness, etc.)
- Complete as atividades interativas

**Opção B: Livro Interativo**
- Clique em "Ler Histórias"
- Selecione uma história
- Leia e responda os exercícios

**Opção C: Quizzes**
- Escolha seu nível (A1, A2, B1)
- Clique em "Continuar Praticando"
- Responda as perguntas

### 4️⃣ Ganhar Pontos

- ✅ Quiz Correto: +10 pontos
- ✅ Atividade Completa: +5 pontos
- 🔥 7 Dias em Sequência: +50 pontos bônus

## 👨‍🏫 Para Professores

### 1️⃣ Fazer Login

1. Acesse a página inicial
2. Clique em **"Entrar como Professor"**
3. Use as credenciais:
   - **Email**: professor@easyway.com.br
   - **Senha**: 123456

### 2️⃣ Explorar o Dashboard

Você terá acesso a:

| Seção | Funcionalidade |
|-------|---|
| **Turmas** | Gerenciar suas turmas |
| **Alunos** | Visualizar lista de alunos |
| **Desempenho** | Ver progresso de cada aluno |
| **Relatórios** | Gerar relatórios de desempenho |
| **Atividades** | Atribuir tarefas |

### 3️⃣ Gerenciar Turma

1. Clique em uma turma
2. Veja a lista de alunos
3. Clique em um aluno para ver detalhes
4. Visualize:
   - Pontos acumulados
   - Nível atual
   - Atividades completadas
   - Progresso geral

### 4️⃣ Atribuir Atividades

1. Vá para "Atividades"
2. Clique em "Nova Atividade"
3. Escolha o tipo:
   - Quiz
   - Leitura (Storybook)
   - Super Skills
4. Selecione a turma
5. Defina a data de entrega
6. Enviar

## 💡 Dicas de Uso

### Para Alunos

1. **Estude regularmente**
   - Dedique 15-20 minutos por dia
   - Mantenha uma sequência de dias (streak)

2. **Comece pelo nível correto**
   - A1 para iniciantes
   - A2 para elementar
   - B1 para intermediário

3. **Use o Super Skills**
   - Não é apenas inglês, é desenvolvimento pessoal
   - Pratique os valores no dia a dia

4. **Leia as histórias em voz alta**
   - Melhora a pronúncia
   - Aumenta a confiança

5. **Não tenha medo de errar**
   - Erros são oportunidades de aprender
   - Tente novamente!

### Para Professores

1. **Acompanhe regularmente**
   - Verifique o progresso dos alunos
   - Identifique dificuldades

2. **Customize as atividades**
   - Escolha conteúdo relevante
   - Ajuste dificuldade conforme necessário

3. **Use o ranking**
   - Crie competição saudável
   - Reconheça os melhores desempenhos

4. **Integre com aulas presenciais**
   - Use as histórias em sala
   - Discuta os valores do Super Skills

5. **Comunique-se com alunos**
   - Dê feedback personalizado
   - Celebre conquistas
