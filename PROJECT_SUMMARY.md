Criar uma plataforma web moderna e interativa para ensino de inglês que combine:
- Educação de qualidade
- Gamificação e diversão

## 🚀 Deployment

### Opções Disponíveis

1. **Netlify** (Recomendado)
   - Drag & drop deploy
   - Deploy automático via Git
   - Domínio personalizado
   - HTTPS gratuito

2. **Vercel**
   - Deploy via CLI
   - Integração com Git
   - Performance otimizada
   - Suporte a serverless

3. **GitHub Pages**
   - Gratuito
   - Integrado com Git
   - Domínio github.io
   - Ideal para projetos open source

4. **Servidor Tradicional**
   - Apache/Nginx
   - Docker
   - Controle total
   - Maior flexibilidade

### Passos Rápidos

```bash
# Netlify
netlify deploy

# Vercel
vercel

# GitHub Pages
git push origin main
# Ativar em configurações
```
